# 🚀 Smart Device Management System

A full-stack real-time device monitoring platform that tracks device status, battery levels, and connectivity using WebSockets.

---

## ✨ Features
- 🔐 JWT Authentication (Login/Register)
- 📱 Device Management (Add, Update, View)
- ⚡ Real-time updates using Socket.IO
- 📊 Live device status (online/offline, battery)
- 🧭 Angular dashboard with routing

---

## 🛠️ Tech Stack
- Frontend: Angular
- Backend: Node.js, Express.js
- Database: MongoDB
- Realtime: Socket.IO
- Auth: JWT
